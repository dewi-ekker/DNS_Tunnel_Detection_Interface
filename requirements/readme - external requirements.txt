external requirements:

Download npcap from https://nmap.org/npcap/#download (also included in this folder).

Download graphviz msi from https://graphviz.org/download/ (also included in this folder) and update Path environment variables: add C:\Program Files\Graphviz\bin to User path and C:\Program Files\Graphviz\bin\dot.exe to System Path. You may need to reboot Windows.

if scapy does not work: follow instructions from https://scapy.readthedocs.io/en/latest/installation.html: download the latest development version (also included in this folder) unzip the archive, open a command prompt in that directory and run python setup.py install
